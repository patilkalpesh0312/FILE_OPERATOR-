## 🌟 About The Project

**Personal Journal Manager** is a simple yet powerful command-line application built with Python that helps users:

✅ Store daily journal entries

✅ View saved memories

✅ Search entries by keyword or date

✅ Delete all entries when needed

✅ Automatically timestamp every journal entry

---

## 🚀 Features

### 📝 Add Journal Entries

Write and save your daily thoughts with automatic timestamps.

### 📚 View All Entries

Display every saved journal entry in an organized format.

### 🔍 Search Entries

Find entries instantly using keywords or dates.

### 🗑️ Delete Entries

Remove all saved entries with confirmation protection.

### ⚠️ Error Handling

Handles:

* File Not Found Errors
* Permission Errors
* Unexpected Exceptions
